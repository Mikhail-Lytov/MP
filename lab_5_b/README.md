# Лабораторная работа N5.6 - Алгоритмы на графах
**Задача:** реализовать следующие функции для работы с графами:
- алгоритм Тарьяна для топологической сортировки;
- алгоритм Флёри;
- алгоритм поиска эйлерова цикла на основе объединения циклов:
алгоритм Косарайю;
Указания по выполнению:
- лабораторная работа состоит из трех файлов: graph-b.h - описание функций: graph-b.cpp - реализация функций; test-graph-b.cpp - тестирование функций;
- формат ввода вывода, структуры для хранения информации, интерфейсы функций выбираются студентом самостоятельно

**Дополнительные задания:**
- осуществить разумный выбор формата ввоца вывода, структур для хранения информации, интерфейсов функций;
